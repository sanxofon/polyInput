# polyInput
 PolyInput is an ES6 modern (JS Client -PHP Server) cross-browser mobile ready online/offline evaluation system.

  - Official font-awesome (4.7.0) resides [here](https://cdnjs.com/libraries/font-awesome/4.7.0)
